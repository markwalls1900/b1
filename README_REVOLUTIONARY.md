## üîß **API Endpoints**

### **Core Endpoints**
- `GET /` - Main dashboard
- `GET /api/entities` - Get all registered entities
- `GET /api/regions` - Get all regional profiles
- `GET /api/matches/{entity_id}` - Get matches for specific entity

### **Response Format**
```json
{
  "matches": [
    {
      "region_id": "TX-AUS",
```
